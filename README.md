## Running the app

```bash

# Clone this repository
$ git clone www.github.com/andres-mateo/graduados-icesi.git

# Go into the repository
$ cd graduados-icesi

# Run the app
$ flutter run -d chrome

# Build the app
$ flutter build web

# Start an http server
$ cd build/web && python -m http.server 8000
```
